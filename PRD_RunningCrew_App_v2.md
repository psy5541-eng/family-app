## 🏁 1. 대시보드 (Dashboard) - 레이스 화면

### 1.1 컨셉
**모든 멤버들이 달리는 레이스 트랙!**
- X축: 마일리지 (거리)에 따른 위치
- Y축: 누적 고도에 따른 높이
- 각자 꾸민 캐릭터가 달리는 애니메이션

### 1.2 화면 구성
```
┌─────────────────────────────────────────────────┐
│  🏁 2026년 3월 크루 레이스        [월간▼] [년간]│
├─────────────────────────────────────────────────┤
│                                                 │
│  ☁️           ☁️                    🏔️ FINISH  │
│                        🏃(철수)──────│ 245km   │
│       ⛰️                             │         │
│            🏃(영희)──────────────────│ 198km   │
│                   🏃(나)─────────────│ 156km   │
│  🌲    🏃(민수)──────────────────────│ 134km   │
│     🏃(지영)─────────────────────────│ 98km    │
│  🌳      🌲                          │         │
│ ════════════════════════════════════│ START   │
│  0km            100km           250km          │
├─────────────────────────────────────────────────┤
│  👆 캐릭터 터치하면 상세 정보 보기               │
└─────────────────────────────────────────────────┘
```

### 1.3 레이아웃 상세

```typescript
type RaceDashboardProps = {
  period: 'monthly' | 'yearly';
  members: RaceMember[];
};

type RaceMember = {
  userId: string;
  nickname: string;
  character: CharacterConfig;      // 꾸민 캐릭터
  totalDistance: number;           // km (X축 위치)
  totalElevation: number;          // m (Y축 높이)
  activityCount: number;
  rank: number;
  isMe: boolean;                   // 내 캐릭터 강조
};
```

### 1.4 캐릭터 위치 계산
```typescript
function calculatePosition(member: RaceMember, maxDistance: number, maxElevation: number) {
  // X축: 마일리지 기준 (0% ~ 100%)
  const xPercent = (member.totalDistance / maxDistance) * 100;
  
  // Y축: 고도 기준 (하단 20% ~ 상단 80%)
  const yPercent = 20 + ((member.totalElevation / maxElevation) * 60);
  
  return { x: xPercent, y: yPercent };
}
```

### 1.5 인터랙션
- **캐릭터 터치**: 해당 멤버 미니 프로필 팝업
- **좌우 스와이프**: 트랙 확대/축소
- **월간/년간 전환**: 기간별 데이터 변경
- **내 캐릭터**: 테두리 강조 + 깜빡임
