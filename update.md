버그 목록
1. 증시 목록 코스피코스닥 그냥 하드코딩된거 뿌리는것 같은 APIKEy 잘되어있는지 확인
2. 밑에 종목도 마찬가지 API KEY 확인
3. 일정 등록/수정시 네이버지도란이 없고 주소검색도 잘안되는것 같은데 이거 내가 APIKEY 네이버맵스만 발급했는데 주소도 따로 필요한지 확인.

* 캘린더
1. 토글 아이콘 추가: 캘린더 헤더(월 표시 옆)에 '달력 뷰'와 '리스트 뷰'를 전환할 수 있는 아이콘 버튼을 추가해 줘.
2. 일정 리스트 컴포넌트: 리스트 뷰를 선택하면 현재 월의 일정들이 시간 순서대로(Chronological) 정렬되어 나오게 해줘.
3. 디자인 반영: 각 리스트 아이템 앞에는 **설정된 색상의 둥근 점(또는 수직 바)**이 표시되어야 하고, 날짜/시간/제목이 깔끔하게 보여야 해.
4. 공비(Empty) 처리: 해당 월에 일정이 없으면 "이번 달은 일정이 없어요. 새로운 추억을 만들어보세요!" 같은 친절한 메시지를 띄워줘.
5. 반응형: 일반적으로는 안드로이드 기본화면에 폴드 같은 넓은 화면에서는 캘린더 옆에 리스트가 동시에 뜨게 하거나, 좁은 화면에서는 탭으로 전환되게 최적화해 줘.
6. 일정 상세에 주소 추가가 되었을때 네이버map연동해서 해당 지도에 토글 찍어줘. 그리고 지도 클릭하면 해당 좌표 기준으로 네이버 지도 앱이나 웹지도로 연동되어야 해.
7. From-To로 일정이 추가될때 캘린더에는 해당 From-To 일자에 맞게 색상바로 그려줘.
7-1. 바안에 일정 제목이 들어가야 함.
7-2 같은 날짜에 여러 일정이 겹치면 위아래 순서대로 쌓이게 (Overlap 처리) 해줘.
7-3 Tailwind활용: 우리 앱 테마에 맞는 부드러운 배경색과 라운딩 처리를 해줘.
